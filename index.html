THE git SITE

git is the most famous version control system currently
GitHub is the most famous platform where the gits are stored. It is the haven for a programmer looking to work with open source or as a team.

  • Create a new GitHub Repository
  • Create a README File containing a description of your project (Optional)
  • Set up your SSH Key and get it authenticated
  • Copy the SSH code of your Repository
  • Open a new project and open up the terminal

  • git clone SSH-Key

    The key can found in the code tab when you open your Repository

  • Make changes to your project as you like
  • Type git status to check the files that have not been updated on git
  • Type git add . to add all the files that have been added or modified to git
  • Type git commit -m "Some title" -m "Some description"
  • Type git push origin main to push the files to your GitHub Repository
  • You are done.
